个人收藏的高质量壁纸，涵盖多种风格（人物、风景、动漫、写实等），分辨率从 2K 到 8K 不等。

## 🖌️ 壁纸分类

### `360/`  
包含大量人物主题壁纸，命名清晰，例如：
- 长发、黑丝、校服、JK制服
- 汉服、女仆、居家、阳光
- 明星壁纸（刘亦菲、倪妮、杨晨晨等）

### `WallhavenDesktop/`  
混合了自然风光、城市、人物及艺术摄影，部分来自 Wallhaven 和 Pexels。
